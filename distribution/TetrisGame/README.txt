俄罗斯方块游戏
================

这是一个使用C++和SFML库开发的俄罗斯方块游戏。

如何运行游戏：
1. 直接双击运行 "Tetri_SFML3.0.exe" 文件即可开始游戏

游戏控制说明：
- A/← : 左移
- D/→ : 右移
- W/↑ : 旋转
- S/↓ : 软降落（加速下降）
- 空格键 : 硬降落（立即下落）
- P : 暂停/继续
- R : 重新开始
- ESC : 退出游戏

游戏特性：
- 经典俄罗斯方块玩法
- 分数系统
- 等级系统（随消除行数增加而加快速度）
- 背景音乐和音效

注意事项：
- 请确保所有文件都在同一目录下
- 不要删除或重命名任何文件，否则游戏可能无法正常运行
- 如果出现任何问题，请确保您的系统支持SFML 3.0运行库

版本信息：
- 开发语言: C++
- 图形库: SFML 3.0
- 平台: Windows

祝您游戏愉快！
